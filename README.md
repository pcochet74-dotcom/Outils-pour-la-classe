# Outils pour la classe
Outils pour la classe est un ensemble de solutions numériques conçues pour faciliter le travail des enseignants et améliorer la gestion de la classe au quotidien. Le projet regroupe plusieurs technologies complémentaires, toutes pensées pour être simples, rapides à utiliser et parfaitement intégrées aux outils déjà présents dans les établissements.

Extensions Chrome / Edge
Ces extensions permettent d’afficher directement, depuis l’ENT ou le navigateur, différents outils pratiques pour la classe. Parmi eux :

un module de mesure du bruit ambiant, utile pour visualiser en temps réel le niveau sonore ;

des outils d’affichage ou de gestion adaptés aux activités pédagogiques ;

des fonctionnalités accessibles en un clic, sans installation complexe.

Applications .NET
Certaines fonctionnalités nécessitent une application locale pour offrir plus de puissance ou d’autonomie. Les applications .NET fournissent notamment :

un système d’enregistrement du volume sonore ;

la conservation et l’exportation des données pour un suivi dans le temps ;

une interface simple, pensée pour être utilisée en classe sans configuration technique.

Compléments Office (COM)
Pour les enseignants qui travaillent dans Microsoft 365, des compléments Office permettent d’accéder aux outils directement depuis :

Word

PowerPoint

Excel

Ces compléments ajoutent un panneau latéral ou des commandes intégrées, afin d’utiliser les outils sans quitter l’environnement de travail habituel.

Outils pour la classe vise ainsi à offrir un écosystème cohérent, léger et polyvalent, qui s’adapte aux pratiques de chaque enseignant. L’objectif est simple : rendre les outils pédagogiques accessibles partout, tout le temps, et sans friction.
